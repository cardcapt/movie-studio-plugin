# movie-studio-plugin

Claude Code で仮想映画制作スタジオを構築・運営するプラグイン。

`/movie` を実行すると制作デスクが窓口となり、あなたの事業に最適化された組織（制作部門）を構築。日常の運営では、制作デスクに話しかけるだけで プロデューサー が適切な制作部門に仕事を振り分けます。

## インストール

```
/plugin marketplace add Shin-sibainu/cc-company
/plugin install company@cc-company
```

## コンセプト

![組織構成図](image.png)

```
あなた → 制作デスク（窓口） → プロデューサー（振り分け） → 各制作部門
```

- **制作デスク**: 常に窓口。何でも相談OK。TODO管理、壁打ち、メモ
- **プロデューサー**: 裏方で判断。制作部門が必要な案件を自動振り分け
- **各制作部門**: 専門領域のファイル管理を担当

ユーザーは制作部門を意識する必要なし。制作デスクに話しかけるだけ。

## 使い方

### 初回セットアップ

```
> /movie

制作デスク: はじめまして！あなたの制作デスクになります。
      事業や活動について教えてください。
あなた: フリーランスのWeb脚本 / 演出やってます

制作デスク: 今の目標を教えてください。
あなた: SaaSを作って月10万の収益を目指してる

制作デスク: 日々困ってることはありますか？
あなた: タスクが散らかる、アイデアを忘れる、クライアント管理が雑

制作デスク: おすすめの制作部門構成はこちらです:
      制作デスク室, 制作進行, 企画脚本 / 演出, 脚本 / 演出, 営業
      いかがですか？
あなた: OK

→ .company/ フォルダが自動生成される
```

### 日常の運営

```
> /movie
制作デスク: おはようございます！何かありますか？

> LP作りたいんだよね、新しいサービスの

制作デスク: 承知しました。各制作部門に指示を出しますね。
  → 制作進行: プロジェクト「新規LP」を作成
  → マーケ: ターゲット・訴求ポイントの整理
  → 脚本 / 演出: 技術要件の洗い出し

> 今日やること教えて

制作デスク: 今日のTODOです:
  最優先:
  - [ ] クライアントAへ見積もり送付
  通常:
  - [ ] LP設計書のラッシュ確認 / 振り返り
  - [ ] ブログ記事の下書き

> ちょっと壁打ちしたいんだけど、料金体系で悩んでて

制作デスク: いいですね！聞かせてください。
      今考えている選択肢はありますか？
```

## 制作部門一覧

| 制作部門 | 担当領域 | 常設 |
|------|---------|------|
| 制作デスク室 | TODO管理、壁打ち、メモ、相談 | はい |
| プロデューサー | 意思決定、制作部門振り分け | はい |
| ラッシュ確認 / 振り返り | 週次・月次ラッシュ確認 / 振り返り | はい |
| 制作進行 | プロジェクト進捗、チケット管理 | |
| 企画脚本 / 演出 | 市場調査、競合分析、技術調査 | |
| マーケティング | コンテンツ企画、SNS、キャンペーン | |
| 脚本 / 演出 | 技術ドキュメント、設計、デバッグ | |
| 経理 | 請求書、経費、売上管理 | |
| 営業 | クライアント管理、提案書 | |
| 美術 | デザインブリーフ、ブランド管理 | |
| 人事 | 採用管理、チーム管理 | |
