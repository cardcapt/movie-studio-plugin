# movie-studio-plugin

Claude Code で仮想映画制作スタジオを構築・運営するプラグイン。

`/movie` を実行すると制作デスクが窓口となり、あなたの事業に最適化された組織（制作部門）を構築。日常の運営では、制作デスクに話しかけるだけで プロデューサー が適切な制作部門に仕事を振り分けます。

## インストール

```
/plugin marketplace add cardcapt/movie-studio-plugin
/plugin install movie-studio@movie-studio-plugin
```

## コンセプト

![組織構成図](image.png)

```
あなた → 制作デスク（窓口） → プロデューサー（振り分け） → 各制作部門
```

- **制作デスク**: 常に窓口。企画相談、TODO整理、メモ、壁打ちを担当
- **プロデューサー**: 裏方で判断。必要に応じて各制作部門へ自動振り分け
- **各制作部門**: 脚本、演出、美術、撮影、編集、音響などの専門領域を担当

ユーザーは制作部門を意識する必要なし。制作デスクに話しかけるだけ。

## 使い方

### 初回セットアップ

```
> /movie

制作デスク: はじめまして！制作デスクです。
            どんな映像作品を作りたいですか？
あなた: 15分くらいのSF短編映画を作りたいです

制作デスク: 今の目標を教えてください。
あなた: 映画祭応募を目指しています

制作デスク: 今困っていることはありますか？
あなた: アイデアはあるけど、脚本の構成とシーン整理がうまくできません

制作デスク: おすすめの制作体制はこちらです:
            制作デスク, プロデューサー, 企画開発, 脚本, 演出, 撮影, 編集
            いかがですか？
あなた: OK

→ .movie-studio/ フォルダが自動生成される
```

### 日常の運営

```
> /movie
制作デスク: おはようございます！今日は何を進めますか？

> 主人公の設定を整理したい

制作デスク: 承知しました。
            企画開発と脚本の観点で整理します。

> 今日やること教えて

制作デスク: 今日の制作TODOです:
  最優先:
  - [ ] ログラインを1案に絞る
  通常:
  - [ ] 第1幕のシーン構成を整理
  - [ ] 参考作品を2本メモ
```

## 制作部門一覧


| 部門 | 担当領域 | 常設 |
|---|---|---|
| 制作デスク | TODO管理、相談受付、メモ、進行整理 | はい |
| プロデューサー | 方針判断、部門振り分け、優先順位整理 | はい |
| レビュー | 日次・週次の振り返り、改善点整理 | はい |
| 企画開発 | ログライン、テーマ、企画整理、参考作品調査 |  |
| 脚本 | プロット、ビートシート、脚本、台詞 |  |
| 演出 | シーン設計、演出意図、ショット方針 |  |
| 美術 | 世界観、ロケーション、衣装、小道具 |  |
| 撮影 | 画角、カメラ、ライティング、映像設計 |  |
| 編集 | 構成、テンポ、シーケンス整理 |  |
| 音響 | BGM、効果音、音響演出、ナレーション |  |
| 配給・PR | タイトル案、紹介文、予告編方針、告知 |  |
